## 2. 핵심 가치

### 2-1. 아침 기분 체크 (Mood Check)
기상 직후 사용자의 현재 기분(무드)을 5종 카드로 확인한다.  
무드는 단순 기록이 아닌 **이후 다이제스트의 톤과 콘텐츠를 결정하는 입력값**이다.

| 무드 | 톤 방향 |
|---|---|
| 개운해요 | 밝고 경쾌하게 |
| 설레요 | 활기차고 풍부하게 |
| 피곤해요 | 차분하고 짧게 |
| 불안해요 | 안정적이고 위로하듯 |
| 그냥그래요 | 중립적, 담백하게 |

### 2-2. 아침 다이제스트 (Morning Digest)
아침 기분에 맞는 톤으로 날씨·뉴스·운세를 **TTS로 음성 브리핑**한다.  
다이제스트 항목은 사용자가 직접 커스텀 가능하다.

| 항목 | 기본 제공 | 커스텀 범위 |
|---|---|---|
| 날씨 | ✅ | 상세도 조절 |
| 뉴스 | ✅ (기본 5카테고리) | 카테고리 선택 (프리미엄) |
| 운세 | ✅ (별자리 기본) | 타로·사주 상세 (프리미엄) |
| 명상·스트레칭 | ❌ | 프리미엄 추가 가능 |

**기상 미션 흐름**: 무드 체크 → 무드 기반 톤 결정 → 커스텀 항목 TTS 다이제스트 → 뉴스 퀴즈

## 3. 게이미피케이션 — 진영 선택

온보딩 시 **다이노진영 / 드래곤진영** 중 하나를 선택한다.

| | 다이노진영 | 드래곤진영 |
|---|---|---|
| 캐릭터 | 공룡 | 드래곤 |
| 톤 | 귀엽고 따뜻함 | 강렬하고 활기참 |
| 스킨 풀 | 다이노 전용 | 드래곤 전용 |
| 리더보드 | 진영별 스트릭 순위 | 진영별 스트릭 순위 |

- 진영 변경 가능하나 스킨·포인트 이전 불가 (변경 억제 설계)
- 진영 간 리더보드 경쟁으로 소속감·재방문율 강화
- 시즌 이벤트에서 진영별 한정 스킨 제공 → FOMO 극대화

> ⚠️ 진영 선택 플로우 및 리더보드 시스템 구체화 필요 (v0.2 미확정)

## 6. 화면 & 기능 상세

### 6-1. 인증 플로우

| 화면 ID | 화면명 | 주요 기능 |

| AUTH-002 | 로그인 | Google / Apple / 이메일 로그인 |

### 6-2. 알람 탭

| 화면 ID | 화면명 | 주요 기능 |
|---|---|---|
| ALM-001 | 알람 탭 (메인) | 알람 카드 목록, ON/OFF 토글, FAB(알람 추가), 공룡 캐릭터(선택 스킨 반영) |
| ALM-002 | 알람 설정 | 시간 피커(드럼롤), 요일 반복, 기상 미션 단계 선택(Stage 1~3), TTS 읽기 항목 선택, 알람음/진동 설정 |

**알람 제한**: 무료 최대 5개 / 프리미엄 최대 20개

### 6-3. 기상 플로우 (알람 울림 → 기상 완료)

```
알람 울림 (WAKE-001)
  ↓ '기상 미션 시작' 탭
Stage 1: 무드체크 (WAKE-S1) — 5종 무드 카드 선택
  ↓ 0.5초 자동 진행
Stage 2: 브리핑 팝업 (WAKE-S2) — 날씨·뉴스·운세 + TTS 재생
  ↓ '다음' 탭
Stage 3: 뉴스 퀴즈 모달 (WAKE-S3) — OX/4지선다, LLM 요약 기반
  ↓ '기상 완료' 탭
기상 완료 (WAKE-END) — 포인트 적립, 스트릭 갱신, 캐릭터 축하 애니메이션
```

| 화면 ID | 화면명 | 주요 기능 |
|---|---|---|
| WAKE-001 | 알람 울림 | 현재 시각 대형 표시, 공룡 캐릭터 애니메이션, '기상 미션 시작' / '5분 더(스누즈)' |
| WAKE-S1 | Stage 1: 무드체크 | 무드 이모지 카드 5종 (개운해요/설레요/불안해요/피곤해요/그냥그래요) |
| WAKE-S2 | Stage 2: 브리핑 팝업 | 날씨 카드, 뉴스 카드, 운세 카드(별자리/사주), TTS 재생 버튼 |
| WAKE-S3 | Stage 3: 뉴스 퀴즈 | 모달 형태, 뉴스 헤드라인+LLM 요약(2~3문장), OX/4지선다, 즉시 피드백 |
| WAKE-END | 기상 완료 | 포인트 자동 적립, 연속 기상 스트릭 표시, '홈으로' 버튼 |

**스누즈**: 최대 3회, 5분 간격  
**미션 스킵**: Stage 3 모달 닫기(X) → 포인트 미지급

## 9. 수익화 모델

### 전환 깔때기

```
무료 사용자
  → 광고 제거 (₩4,900 일회성) — 첫 결제 진입점
  → 프리미엄 구독 (₩4,900/월) — 핵심 수익원
  → 연간 구독 (₩39,900/년, 약 33% 할인) — LTV 극대화
  + IAP 병행 — 비구독자 수익화
```

### 리텐션 루프 3중 설계

1. **기상 스트릭** — 연속 기록을 깨기 싫어서 매일 실행
2. **포인트** — 기상 완료 시 자동 적립, 스킨 구매로 소진
3. **스킨/캐릭터 성장** — 캐릭터 애착 → 지속 참여

### 바이럴 설계

- 친구 챌린지 초대 시 2주 프리미엄 무료 체험 제공
- 진영별 리더보드 경쟁으로 자연 공유 유도
- 시즌 한정 스킨 FOMO 마케팅
